# プロパティ(Property)～性質、属性～

## button1のプロパティを観察する

- BackColor
  - 背景色
- Enabled
  - true=有効　False=無効
- Font
  - 書体や文字の大きさ
- ForeColor
  -文字の色
- Location
  - Controlの位置。プログラムからはLeftとTop
- Size
  - コントロールの大きさ。プログラムからはWidth,Height;
- Text
  - 表示する文字
- TextAlign(テキストアライン)
  - 文字の配置
- Visible
  -  true=表示　False=非表示

他にも、BackgroundImage, FlatStyle, Image, ImageAlignなど、時間がある時にいじってみよう。
